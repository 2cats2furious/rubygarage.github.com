The basic types

Numbers

Native integers are represented by the Fixnum class. It's called Fixnum because it has a fixed size. They're fast, the computer knows how to work with them and they're everything more people will ever need.

When an operation between two Fixnum objects creates an overflow, Ruby will automatically convert the numbers to a Bignums, perform the operation again and store the result in a Bignum. The Bignum is returned from the operator and stored in the variable just like a Fixnum would be. The programmer doesn't have to know the Bignum even exists.

Mixins

module Calculate

  def to_cents(dollars)
    Integer(dollars * 100 + 0.5)
  end

  def to_dollars(cents)
    cents / 100.0
  end
end

class BookInStock

  attr_reader :titile, :author, :price
  attr_accessor :price

  include Calculate

  def initialize(title, author, price)
    @title = title
    @author = author
    @price = Float(price)
  end

  def price_in_cents
    to_cents @price
  end

  def price_in_cents=(cents)
    @price = to_dollars(cents)
  end
end
